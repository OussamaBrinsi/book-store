# Book Store

A full-stack book store web application with React (frontend) and Node.js/Express (backend).

## Features

- User authentication and registration
- Browse, search, and purchase books
- Admin dashboard for managing books, orders, and users
- Order management and statistics
- Responsive UI with Tailwind CSS

## Project Structure

- `frontend/` — React app (Vite, Tailwind, Redux)
- `backend/` — Node.js/Express REST API

## Getting Started

### Backend

1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```
2. Start server:
   ```bash
   npm start
   ```

### Frontend

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## Deployment

- Both frontend and backend are configured for Vercel deployment.
